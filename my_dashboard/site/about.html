This is an example site to demonstrate how a shiny dashboard can be integrated into a full-service reporting system. Some pages are shiny only pages while others are a mix. Some pages, like this one, have no shiny app at all. If you have existing reporting you can integrate it into one site. In general, Shiny will be unable to directly handle terabytes of data. With the proper pipeline, however, terabyte data sources can be condensed in a way that allow dashboards to run indirectly against them.

Praxis Data Science created some pages to demonstrate some capabilities. Some pages, however, are other peoples' examples. We do our best to give credit where it is do, please let us know if we missed someone.
